## 진행 방식

- **기간**: 2025.11.20 ~ 2026.01.15 (8주)
- **일시**: 매주 목요일 오전 11:00 ~ 12:30
- **발표**: 매주 1명이 해당 주차 내용 발표
- **정리**:
  - 개인별 정리: 스터디 전까지 작성
  - 통합 정리: 차주 화요일까지 완료
